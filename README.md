# Easy Mine Game in C
 Easy Mine Game in C

## Compile and Run Code
gcc main.c -o main  
./main

## Execute Code
Generate a NxN map with M bombs  
Input step location row, col. If step on bomb then Game Over, else step place will spread out none bomb area in direction (up, left, right, down).
